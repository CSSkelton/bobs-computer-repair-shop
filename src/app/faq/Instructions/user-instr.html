User Guide

Sign In Information

  1. Please enter your assigned user name in the field labeled "User Name".

  2. Please enter in your password in the field labeled "Password".


  3. After you have correctly entered in all the required information. Please select the button labeled "Sign In".


Any information that is not correctly enter in will display a "Sign In Error Message" .

If you are unable to Sign In. Click on Reset Password Guide from the Frequently Asked Questions Page











Edit User

  1. Please enter in the employee email address in the select field name "Email.

  2. Enter in the first name in the select field name"First Name".


  3. Enter in the last name in the select field name"Last Name".


  4. Enter in the phone number in the select field name"Phone Number".


  5. Enter in the home address in the select field name"Address".

Please make sure to click on the "Update button" in order to save all changes.
