User Guide

Sign In Information

  1. Please enter your assigned user name in the field labeled "User Name".

  2. Please enter in your password in the field labeled "Password".


  3. After you have correctly entered in all the required information. Please select the button labeled "Sign In".


Any information that is not correctly enter in will display a "Sign In Error Message" .

Proceed Below to Reset Password











Password Reset

  1. In order to obtain a new password you will need to answer all the following questions with the correct information. Please select the field label "Question 1".

  2. After correctly answer the first question. Please proceed on to the field labeled "Question 2".


  3. The final question you will need to answer will be labeled "Question 3".


  4. This process can not be completed with missing data.After you have correctly entered in all the required information. Please select the button labeled "Reset".


You will be prompted to create a new password and be eligible to log back into the system.
